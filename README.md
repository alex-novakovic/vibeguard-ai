# vibeguard-ai
An AI agent that stops you from tinkering and makes you ship. VibeGuard runs a scoping session, tracks time and token usage per feature, detects drift from your product vision, and suggests the next task — so you finish what you started.

## Setup

### Environment variables

Create a `.env` file in the project root (it is gitignored and should never be committed):

```
ANTHROPIC_API_KEY=your_api_key_here
```

Get your API key from [console.anthropic.com](https://console.anthropic.com).
